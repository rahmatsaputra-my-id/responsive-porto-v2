UI & UX Design

UX designer thinks about how the experience makes the user feel, and how easy it is for the user to accomplish their desired tasks. They also observe and conduct task analyses to see how users actually complete tasks in a user flow. UI designer will think about icons and buttons, typography and color schemes, spacing, imagery, and responsive design.

Website Developer

Web developer is someone who takes a web design - created by a client or design team - turning it into a website. They do this by writing complex lines and lines of code, using multiple languages. Web developers have a pretty tough job, because they basically have to take a language we understand, like English, and translate it into a difficult language, like Java or HTML.

Mobile App Developer

Mobile app developers are programmers who are accustomed and trained to build products in the form of mobile-based applications. The work they do is based on design ideas as well as software engineering applications. The task of a mobile app developer is very different from that of a web developer.
